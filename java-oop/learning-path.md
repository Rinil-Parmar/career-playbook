# Java OOP Learning Path (Co-op + Interview Focus)

This track is designed in an industry-friendly order so each concept builds on the previous one.

## Phase 1: Foundation

1. Java syntax refresh (types, loops, methods, arrays)
2. Classes and objects
3. Encapsulation and access modifiers
4. Constructors and `this`

## Phase 2: Core OOP

1. Inheritance and method overriding
2. Polymorphism (compile-time and runtime)
3. Abstraction (`abstract` classes and interfaces)
4. Composition vs inheritance (real design decisions)

## Phase 3: Interview-Critical Topics

1. `static`, `final`, immutability
2. `equals()` and `hashCode()`
3. Object lifecycle and memory basics
4. Exception design and robust method contracts

## Phase 4: Practical Engineering

1. Collections + generics in object-oriented design
2. Small domain model design (clean class responsibilities)
3. Refactoring bad OOP examples
4. Mini project (extensible, testable, interview-ready)

## Study Method Per Lesson

1. Learn concept in 10-20 minutes
2. Read one clean example
3. Solve one interview-style coding prompt
4. Explain the concept in your own words

## Deliverables for This Track

1. Reusable Java OOP notes
2. Solved interview question bank
3. Mini project you can show in co-op applications
