## Quick Summary

**Top 5 skills to learn before Fall 2026 applications:**
1. Python + ML basics
2. LLM APIs + Prompt Engineering
3. RAG pipelines
4. Cloud + AWS basics
5. Docker + Kubernetes

**Buzzword to know right now:** MCP (Model Context Protocol) — the new standard for connecting AI agents to tools. [Read more →](mcp/what-is-mcp.md)

**Target domain:** Fintech / Banking (RBC, TD, BMO, Scotiabank, fintech startups)
